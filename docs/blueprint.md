# **App Name**: Brutal Score

## Core Features:

- Test Configuration & PDF Upload: Users can upload test papers as PDFs, define the number of subjects and sections per subject, and set specific marking schemes (positive/negative/partial) for each section.
- Dual-Panel Test Interface: A split-screen layout with a resizable PDF viewer on the left for test paper display and a 'Response Console' on the right.
- Dynamic Response Console: The right panel dynamically adapts input controls (radio buttons, checkboxes, numeric input) based on the question type defined in the test configuration.
- Integrated Test Timer & Navigation: A visible timer for the entire test duration, alongside a scrollable grid of question numbers for quick navigation between questions.
- Post-Test Answer Key Input: After test submission, a dedicated interface to input the correct answers for all questions to facilitate automatic scoring.
- Score & Basic Performance Calculation: A core function to calculate total score, incorporating positive, negative, and partial marking based on user responses and the provided answer key.
- Foundational Analytics Display: Present key metrics like overall accuracy, subject-wise accuracy, and total score after calculation, setting the stage for deeper analysis.

## Style Guidelines:

- Primary interactive color: A vibrant, clear blue-violet (#6B88FF) to signify actions and focal points against a dark theme, embodying focus and intellect.
- Background color: A deep, muted dark grey-blue (#242427) providing a distraction-free, professional dark canvas.
- Accent color: A bold cyan-blue (#008CCC) used for highlighting critical information, warnings, or secondary actions, creating effective contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and neutral appearance, ensuring maximum readability and alignment with a data-focused, 'brutal transparency' aesthetic.
- Utilize minimalist, crisp line icons that are clearly identifiable and functionally intuitive, maintaining the clean and distraction-free visual design.
- Strict adherence to a responsive split-screen layout for desktop, ensuring the PDF viewer and Response Console are always visible and logically structured.
- Subtle, quick animations and transitions to provide feedback for user interactions without causing visual distractions or delays, maintaining a high-performance feel.
